Medical Expert System / Health Advisor

A modern web application that provides health-related suggestions based on user input.  
Built with a clean UI and fast frontend architecture.

Features
- User-friendly interface
- Health advice based on input symptoms
- Fast and responsive design
- Modern UI with Tailwind CSS

Tech Stack
- TypeScript
- Vite
- Tailwind CSS
- HTML / CSS / JavaScript

Project Structure
- `src/` → main application code  
- `public/` → static files  
- `index.html` → entry point
